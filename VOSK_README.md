# Vosk 离线模型下载说明

如果你想使用 **Vosk 离线识别模式**（无需网络连接），请按照以下步骤下载并安装语音模型：

## 下载步骤

1. **访问 Vosk 模型下载页面**
   - 网址：https://alphacephei.com/vosk/models

2. **选择中文模型**
   - 推荐：`vosk-model-cn-0.22`（约 50MB）
   - 或者：`vosk-model-small-cn-0.22`（约 42MB，更小但准确率略低）

3. **下载并解压**
   - 下载 `.zip` 文件
   - 解压到本程序目录下
   - 将解压后的文件夹重命名为 `vosk`

4. **验证目录结构**
   确保目录结构如下：
   ```
   程序根目录/
   ├── vosk/
   │   ├── am/
   │   ├── conf/
   │   ├── graph/
   │   └── ivector/
   ├── main_modular.py
   └── ...
   ```

## 使用方法

1. 打开程序
2. 进入"设置"Tab
3. 选择"💾 Vosk 离线识别"
4. 返回"主界面"Tab
5. 点击"开始监听"

## 注意事项

- Vosk 模型文件较大（约 50MB），不包含在程序发布包中
- 首次使用需要手动下载
- 离线模式无需阿里云账号，完全本地运行
- 识别准确率略低于阿里云在线模式，但无需网络连接

## 常见问题

**Q: 提示"模型路径不存在"？**
A: 请确认 `vosk` 文件夹在程序根目录下，且包含 `am`, `conf`, `graph` 等子文件夹。

**Q: 提示"未安装 vosk 库"？**
A: 运行命令：`pip install vosk`

**Q: 识别效果不好？**
A: 尝试下载更大的模型（如 `vosk-model-cn-0.22`），或切换到阿里云在线模式。
